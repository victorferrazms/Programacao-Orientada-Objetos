# ☕ Programação Orientada a Objetos (POO)

Repositório dedicado às resoluções de exercícios práticos da disciplina de **Programação Orientada a Objetos**, do curso de Tecnologia em Análise e Desenvolvimento de Sistemas (TADS) no **Instituto Federal do Tocantins (IFTO) - Campus Araguaína**.

## 🎯 Objetivo

Praticar a sintaxe da linguagem Java e o desenvolvimento de algoritmos para a resolução de problemas lógicos e matemáticos, estabelecendo a base rigorosa necessária para a compreensão dos pilares da Orientação a Objetos.

## 🗂️ Estrutura do Projeto e Exercícios

O repositório está dividido em diretórios de acordo com as listas de atividades desenvolvidas ao longo do semestre.

### 📂 Lista 01: Cálculos e Operações Básicas
Foco em entrada e saída de dados (`java.util.Scanner`), manipulação de variáveis `Double` e cálculos matemáticos essenciais.
* **Ex01:** Operação de potenciação utilizando a classe `Math` para calcular o cubo de um número real ($x^3$).
* **Ex02:** Algoritmo de conversão cambial (Real para Dólar) a partir da soma de valores de entrada.
* **Ex03:** Cálculo de salário com aplicação de reajuste percentual (15%).
* **Ex04:** Operações com múltiplos valores reais para acréscimos (30%) e descontos (25%).
* **Ex05:** Lógica geométrica para cálculo de área de terrenos retangulares utilizando a fórmula $A = b \cdot h$.
* **Ex06:** Lógica de proporção aplicada ao cálculo de ferraduras necessárias para os cavalos de um haras.

### 📂 Lista 02: Lógica Aplicada a Cenários Reais
Aplicações de regras de negócios simples, automação de cadastros e equações matemáticas.
* **Ex01:** Sistema automatizado para uma padaria (cálculo de faturamento diário com pães e broas, incluindo a extração de 10% para poupança).
* **Ex02:** Algoritmo de conversão cronológica (transformação de idade em anos completos para dias de vida).
* **Ex03:** Lógica de abastecimento de combustível (cálculo de litros obtidos a partir de um valor financeiro estipulado).
* **Ex04:** Estrutura base de classe e método principal em Java.
* **Ex05:** Resolução algorítmica da função matemática $x^2 + 4x + 2$.
* **Ex06:** Formulário de cadastro funcional via terminal (recebimento e impressão de nome, idade e telefone de funcionários).

## 🛠️ Tecnologias Utilizadas

* **Linguagem:** Java
* **Recursos:** Uso intensivo da classe `Scanner` para interação via console e da classe nativa `Math` para cálculos complexos.
